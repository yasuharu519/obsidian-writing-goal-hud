# Writing Goal HUD

目標文字数までの進捗を、Obsidianのアクティブなエディタペイン右下にHUD表示するプラグインです。日本語文字数のカウントに対応しています。

```
┌─────────────────┐
│ 全体 312 / 1000 字 │
│ ████░░░░░░░░░░░░ │
└─────────────────┘
```

- 選択範囲があれば「選択」、なければ「全体」を表示
- 目標を超えるとプログレスバーの色が変わります
- HUD はアクティブなマークダウンペイン内に追従するので、ファイルツリーなど他ペインがアクティブな時は表示されません
- 右上の `▾` ボタンで最小化、`▸` ボタンで展開できます (状態は保存されます)

## 目標文字数の設定

優先順位:

1. **ノートの frontmatter** に `writing_goal` を書くと、そのノートだけ目標を上書きできます

    ```yaml
    ---
    writing_goal: 2000
    ---
    ```

2. **設定タブ → Writing Goal HUD → デフォルト目標文字数** で全体のデフォルトを変更できます (初期値: 1000)

## カウント仕様

- コードブロック (` ``` ` で囲まれた部分) を除外
- マークダウンリンク・画像 `[text](url)` `![alt](url)` を除外
- Obsidian Wikiリンク `[[note]]` を除外 (`[[note|表示]]` の表示部分は残す)
- 自動リンク `<https://...>` を除外
- 生のURL `https://...` を除外
- マークダウン記号 `# * _ > \` ~ - [ ] ( ) | \\` を除外
- 全空白文字を除外
- 文字数は `[...str].length` で数える (サロゲートペア対応)

## インストール

### 手動インストール

1. `pnpm install` で依存関係をインストール
2. `pnpm build` で `main.js` を生成
3. Obsidian Vault の `.obsidian/plugins/writing-goal-hud/` ディレクトリを作成
4. 以下の3ファイルをそのディレクトリにコピー
    - `main.js`
    - `manifest.json`
    - `styles.css`
5. Obsidianの設定 → コミュニティプラグイン → Writing Goal HUD を有効化

### 開発用

リポジトリディレクトリそのものを Vault に貼ると、リビルドが即反映されて便利です:

```bash
ln -s /path/to/this/repo \
      /path/to/your/Vault/.obsidian/plugins/writing-goal-hud
pnpm install
pnpm dev
```

`pnpm dev` は esbuild の watch モードで `main.js` を更新し続けます。`main.js` 更新後にプラグインを Obsidian で再読み込みすると反映されます (`pjeby/hot-reload` プラグインを入れておくと自動)。

## 将来の拡張余地

今後追加するかもしれない機能:

- 単語数カウント (英語向け)
- カウント方式の切替 (記号込み / 除外、コードブロック含む / 含まない)
- 表示位置切替 (四隅)
- IME composing 中のスキップ (CM6 の updateListener 経由)
- 原稿用紙換算枚数の併記オプション (400/200字換算)
- 「ゴールまであと N 字」表示モード
